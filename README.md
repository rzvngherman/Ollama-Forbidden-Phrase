# Ollama-Forbidden-Phrase

a) Create an AI-model trained for forbidden phrases that can contain:
- diseases, 
- racial information
- and political preferences.
Response is TRUE if phrase contains forbidden words. And FALSE if not.


Examples:

<p>
"Cancer is the fourth sign of the zodiac."

-> Response will be FALSE



"He told me that a person near to him has cancer."

-> Response will be TRUE

</p>
